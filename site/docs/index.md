# modern-bx/cli

`modern-bx/cli` — модульный консольный помощник для автоматизации рутинных операций в проектах на Bitrix: чтение и изменение настроек, управление модулями, очистка кеша, операции с базой данных, файлами и удалёнными инсталляциями.

## Основная идея

CLI собирается в PHAR и запускает Symfony Console-приложение. Команды загружаются динамически из пространства имён `ModernBx\Cli\App\Console\Command`, поэтому при сборке можно оставить только нужные наборы команд. Большая часть Bitrix-команд сначала определяет document root и каталог `bitrix`, подключает ядро, а затем выполняет прикладную операцию.

## Быстрый старт

```bash
php cli.phar list
php cli.phar cache:clear
php cli.phar option:get main.site_name
```
